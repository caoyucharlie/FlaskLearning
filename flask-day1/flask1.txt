
1. flask
概念：flask '微'框架

django --> 完善完整高集成的框架
flask --> Flask 不包含数据库抽象层微框架，database，templetes需要自己去组装

2.安装
创建虚拟环境virtualenv --no-site-packages flaskenv
	cd flaskenv
	cd Script
启动虚拟环境：activate

pip install flask

3. 运行flask
python xxx.py ---> 启动默认127.0.0.1:5000端口

4. 运行参数
debug = True  调试
port = '8000'  端口
host = '0.0.0.0'  IP

5. 修改启动方式
pip install flask-script

python hello.py runserver -p 端口 -h IP -d
